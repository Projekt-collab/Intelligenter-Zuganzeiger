Echtzeit-Stellwerk-Steuerung

Die Lokomotive steht bereit.
Warten auf die richtige Fahrstraße.

Herzlich willkommen beim „Intelligenten-Zuganzeiger“! Dieses Projekt simuliert eine automatisierte, digitale Stellwerk-Steuerung in Echtzeit на базе Django, Redis и WebSockets.

Der Zug am Bahnsteig symbolisiert den Systemstart: Wenn eine Lokomotive einfährt, fordert sie vollautomatisch ein freies Gleis an. Das Stellwerk prüft die Sensoren, aktiviert die Kollisionserkennung und reserviert die Fahrstraße für absolute Sicherheit.

⚙️ System-Architektur & Protokoll-Abstraktion

Django & MySQL

Django wird für die Hauptlogik genutzt, MySQL für die dauerhafte Speicherung der Fahrpläne und Zugdaten.

Redis & WebSockets

Redis fungiert als Message Broker für die WebSockets zur Datenübertragung in Echtzeit.

JSON Protokoll

Flexible Architektur, um nicht an feste Hardware gebunden zu sein, mittels JSON über WebSockets.

🛡️ Device-Sicherheit & Kollisionserkennung

Die mathematische Absicherung unseres Systems basiert auf einer strikten Regel: Gleisstatus == 0. Ein Gleisabschnitt wird erst und nur dann für die Einfahrt freigegeben, wenn die Sensoren absolute Freiheit melden. Dies verhindert Fehlsteuerungen im Stellwerk UI komplett.
